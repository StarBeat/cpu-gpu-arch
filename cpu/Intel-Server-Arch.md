

# Sapphire Rapids (2023)
Server CPU.<br/>
**Golden Cove** microarchitecture.

## Examples

* Xeon Max 9480
* Xeon Platinum 8400
* Xeon Gold 5400, 6400
* Xeon Silver 4400

## Notes

* AVX512 extensions: F, CD, VL, DQ, BW, IFMA, VBMI, VBMI2, BITALG, VNNI, GFNI, VPOPCNTDQ, VPCLMULQDQ, VAES, BF16, FP16
